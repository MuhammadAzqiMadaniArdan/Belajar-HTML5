3.viewBox (jendela)
->jendela svg tag dalam svg akan menjauh atau mendekat
viewBox = ("x minimal , y minimal, width, height")
Ket :
x -> posisi Horizontal 
contoh :
x = 50
maka jendela akan berpindah ke samping
y -> posisi vertical
width -> besar lebar jendela
height -> seberapa tinggi jendela

4.Width dan height dari svg adalah view port
